onReady(callback)

Sets callback function which will invoked when all classes of the module will be loaded (if configuration parameter "autoload" was set in true) and registered. It is the same as "onReady" parameter in module configuration. If it was defined in module configuration too the new callback function will be added to the onReady callbacks storage and will be invoked after other callback functions which were registered earlier. If "autoload" configuration parameter was set in false and there were no classes registered in module at the moment and onReady callback function was not set earlier, the call of current method invokes specified callback immediately.
Parameters:
Name Type Description
callback function Callback function which will do some initializing manipulations
